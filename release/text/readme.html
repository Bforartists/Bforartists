Bforartists



About

Welcome to the 3D graphics software Bforartists.

Bforartists is a complete 3D suite to create 3D content. It offers you the full 3D art pipeline to create game graphics, prerendered movies and stills. From modeling, sculpting, texturing, rigging, animation, rendering, up to post processing. On top of that it offers also a video editor. Have a look at the features page to find out more.

Bforartists can be used freely for any purpose, including commercial use and distribution. Bforartists is free and open-source software, released under the GNU GPL licence. You can find the sourcecode at Github:

https://github.com/Bforartists/Bforartists



Bug reporting

Every software has bugs. And while we always hope that no bugs makes it to the user, there's always something going through.

When you find a bug then you can report it in our bugtracker: https://github.com/Bforartists/Bforartists/issues

Package Contents

• The Bforartists application for the chosen operating system and version.

• Scripts for importing and exporting to other 3d formats.

• Readme and copyright files.



Installation

Windows: Download the installer, start the installation, and follow the instructions of the installer.

There is also a zipfile version available. The archive contains a Bforartists folder. You may put this anywhere on your hard drive. To launch Bforartists, double-click on Bforartists.exe.

Linux: There is no Linux binary available yet. You have to compile the application from source. Here's a short tutorial that shows the whole process under Linux Ubuntu 17 https://www.bforartists.de/data/tuts/Building_with_Ubuntu17.pdf

Mac OS X: Same as with Windows. Download and install.

Installing Addons (all systems) Addons can be installed from the user preferences addons section, download an addon as a .py or .zip file, then press the "Install Addon" button and select the file to install it.
